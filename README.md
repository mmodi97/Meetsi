## Meetsi

A Free Open Sourced Video Calling Platform.<br>
Video Conferencing by Jitsi Library<br>
Designed in Kotlin<br>


### Used Libraries
- Koin Dependency Injection
- MVVM Arch.
- Kotlin Extenstions
- Room Database
- LiveData
- RxJava
- Coroutines


### Special Thanks

Jitsi Library for Free and Open Source Api Provider
